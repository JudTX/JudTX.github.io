    rgba()和hsla()

    RGBA指的是“红色、绿色、蓝色和Alpha透明度”(Red-Green-Blue-Alpha)

    HSLA则代表“色调、饱和度、亮度和Alpha透明度”(Hue-Saturation-Lightness-Alpha)
